# 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar e demonstrar conhecimentos em:

- Desenvolvimento de aplicações web com **Flask**
- Organização de projetos backend
- Estruturação de **rotas e templates**
- Autenticação básica de usuários
- Separação entre frontend e backend
- Criação de sistemas web simulando aplicações reais

Projeto desenvolvido para fins de aprendizado em **desenvolvimento web com Python e Flask**.
